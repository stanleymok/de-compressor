# de-compressor
Software to compress / decompress files in Java

## to run
## to compile
```
javac *.java
```
### run
#### compress
```
java Main <folder-to-be-compressed> <output-folder> <max-compressed-file-in-MB>
```
#### to decompress
```
java Main <folder-to-be-decompressed> <output-folder>
```
